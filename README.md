# SwrpgApp

## DATABASE SETUP

Using: Postgresql version 10.10-2

**Setting Up the database:**

1. Create a new database called nwen304_groupproject:
    1. `psql -U <user>`
    2. `CREATE DATABASE swrpgdb;`
2. Create all the tables using `psql -U <user> -d swrpgdb -f ~/database.data`
